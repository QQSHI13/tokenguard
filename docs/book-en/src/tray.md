# Tray icon & notifications

The tray icon color reflects the most critical active limit:

- **Green** — everything is within limits.
- **Yellow** — a limit has crossed 80% of its cap.
- **Orange** — the proxy is paused.
- **Red** — a limit has been exceeded.

## Actions

- **Left-click** — pause or resume the proxy.
- **Double left-click** — open the Dashboard.
- **Right-click** — open the menu: Dashboard, Settings, pause/resume, quit.

Token Guard also shows desktop notifications when a limit warning or block occurs, throttled so it does not spam.
