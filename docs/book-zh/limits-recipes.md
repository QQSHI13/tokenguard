# 限额配方

限额是本地计数器，在超出上限前进行警告、阻止或暂停。所有周期均按 UTC 计算。

## 每日预算

防止意外超支。

| 字段 | 值 |
|---|---|
| 指标 | 金额 |
| 上限 | `50` |
| 周期 | 每天 |
| 范围 | 全局 |
| 动作 | 阻止 |
| 警告阈值 | `0.8` |

结果：今日支出达到 $50 后，Token Guard 返回 HTTP 429。

## 月度令牌额度

模拟每月包含 100 万令牌的套餐。

| 字段 | 值 |
|---|---|
| 指标 | 令牌 |
| 上限 | `1000000` |
| 周期 | 每月 |
| 范围 | 全局 |
| 动作 | 警告 |
| 警告阈值 | `0.9` |

结果：超过 90 万令牌时显示托盘通知；请求继续通过。

## 每小时请求上限

保护共享密钥或有速率限制的端点。

| 字段 | 值 |
|---|---|
| 指标 | 请求 |
| 上限 | `100` |
| 周期 | 每小时 |
| 范围 | 服务商 |
| 范围 ID | 你的服务商 |
| 动作 | 阻止 |

结果：该服务商本小时达到 100 次请求后，新请求返回 429。

## RPM 限制

防止突发流量触发服务商自己的速率限制。

| 字段 | 值 |
|---|---|
| 指标 | 每分钟请求数 |
| 上限 | `60` |
| 周期 | —（RPM 始终使用 60 秒滚动窗口） |
| 范围 | 全局 |
| 动作 | 阻止 |

结果：任意 60 秒窗口内不超过 60 次请求。

## 时间上限

用于类似"每天 5 小时"的订阅套餐。

| 字段 | 值 |
|---|---|
| 指标 | 时间（秒） |
| 上限 | `18000`（5 小时） |
| 周期 | 每天 |
| 范围 | 全局 |
| 动作 | 暂停 |

结果：累计请求时长达到 5 小时后，代理暂停，直到你手动恢复。

## 项目专属预算

防止单个项目耗尽整个团队预算。

| 字段 | 值 |
|---|---|
| 指标 | 金额 |
| 上限 | `20` |
| 周期 | 每天 |
| 范围 | 项目 |
| 范围 ID | 你的项目 |
| 动作 | 阻止 |

结果：带有该项目标签的请求今日达到 $20 后被阻止。

## 计划限额

使用**生效时间**和**生效日期**让限额只在工作时间或工作日生效。时间为 UTC。

示例：在工作日 09:00 到 17:00 UTC 之间，阻止支出超过 $100。

| 字段 | 值 |
|---|---|
| 指标 | 金额 |
| 上限 | `100` |
| 周期 | 每天 |
| 范围 | 全局 |
| 动作 | 阻止 |
| 开始时间 | `09:00` |
| 结束时间 | `17:00` |
| 生效日期 | 周一至周五 |
