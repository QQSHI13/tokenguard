> 本地 LLM 网关。你的密钥、你的机器、你的令牌。

Token Guard 位于你的编码代理和 LLM API 之间。它把元数据（词元、模型、费用）记录到本地 SQLite 数据库，并在系统托盘中显示实时花费。

**3 × 3 格式转换。** Token Guard 接受 OpenAI、Anthropic 或 Google Gemini SDK 形状的请求，并把它们转发给任意已配置的服务商，按需转换请求体、响应体和流式片段。你可以继续使用已有的 SDK 调用想要的服务商。

**除了你主动发送的 API 调用，没有任何东西离开你的机器。** Token Guard 没有云账号、没有遥测、没有外部计费服务。API 密钥保存在系统钥匙串中，不会写入磁盘。
